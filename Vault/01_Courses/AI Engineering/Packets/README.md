# AI Engineering Packets

Packet source files for NotebookLM should live here by default.

This folder follows the GitHub-first packet workflow:
- packet files are curated teaching documents,
- packet files stay near the course they belong to,
- packet files are reviewed in GitHub,
- and exports for NotebookLM are downstream artifacts rather than the primary notes.
