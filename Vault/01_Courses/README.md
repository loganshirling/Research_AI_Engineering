# 01_Courses

This folder contains course-level learning paths.

A course should:
- define a clear learning objective
- provide an ordered path through major sections
- link to supporting topic, concept, technique, practice, and review notes
- remain understandable in plain GitHub markdown and plain Obsidian

Recommended pattern:
- one course folder per major subject
- one course home note
- one roadmap note
- optional packet and episode subfolders
