## How to use this folder

Use topic notes when you need a practical working model for one major subdomain such as retrieval, serving, evaluation, agents, or multimodal systems.

A good rhythm is:
1. read the relevant section in [[AI Engineering - Roadmap]]
2. read one topic note here
3. review linked concepts or techniques
4. do a related practice drill
5. return to [[Map - AI Engineering]]
