## Vault structure

The repo is organized as a plain-markdown learning vault:

- `Vault/01_Courses/` course homes, roadmaps, packets, and episode notes
- `Vault/02_Topics/` topic notes
- `Vault/03_Concepts/` concept notes
- `Vault/04_Techniques/` repeatable workflows and checklists
- `Vault/06_Sources/` source summaries and evidence anchors
- `Vault/08_Practice/` drills and build notes
- `Vault/09_Reviews/` checkpoint reviews and retention notes
- `Vault/10_Maps/` map notes for navigation
